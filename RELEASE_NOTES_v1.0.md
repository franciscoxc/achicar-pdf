# Achicar PDF v1.0

Primera version publica.

## Incluye

- App Windows portable en un solo ejecutable.
- Version x64 y ARM64.
- Drag and drop de PDFs.
- Modo inteligente con tamano maximo en MB.
- Modo manual con PPI.
- Opcion para forzar gris.
- Descarga guiada de Ghostscript desde el GitHub oficial de Artifex.
- Resultado en la misma carpeta del original.

## Binarios

- `Achicar PDF x64.exe`
- `Achicar PDF arm.exe`

## Nota sobre Ghostscript

Achicar PDF no incluye Ghostscript. Si no esta instalado, la app descarga el
instalador oficial de Artifex. Ghostscript tiene licencia propia.
